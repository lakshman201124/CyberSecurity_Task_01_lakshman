# Part C: Online Account Security Check

## Account Security Summary

| Account | MFA Enabled? | Strong Password? | Recovery Email Configured? |
|---|---|---|---|
| Google | Yes | Yes | Yes |
| GitHub | Yes | Yes | Yes |
| Instagram | Yes | Yes | Yes |

## Notes / Observations
- All three accounts reviewed had MFA enabled, strong passwords in use, and a recovery email configured.
- This audit confirmed that baseline account security is in good shape across the platforms checked.
- Ongoing good practice: periodically verify that the recovery email linked to each account is still active and accessible, since an outdated recovery email can lock you out in a crisis.

## How to Check
- **Google**: myaccount.google.com → Security tab
- **GitHub**: Settings → Password and authentication
- **Instagram**: Settings → Accounts Center → Password and security
